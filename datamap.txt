tables in resnet 2021-Aug 13


attr  - name,value pair attributes that are connected with the other data type
                Table "resnet.attr"
 Column |  Type   | Collation | Nullable | Default 
--------+---------+-----------+----------+---------
 id     | bigint  |           | not null | 
 name   | text    |           |          | 
 value  | text    |           |          | 
 index  | integer |           |          | 

	id - unique 64bit integer id
	name - text, name of attribute
	value - text, value of attribute
	index - integer. when an item has a list of attributes this index groups them together. that is, 
			the year, title, etc. of each reference have the same index


node - each record is an entity used in a relationship. 
                  Table "resnet.node"
   Column   |   Type   | Collation | Nullable | Default 
------------+----------+-----------+----------+---------
 id         | bigint   |           | not null | 
 urn        | text     |           |          | 
 attributes | bigint[] |           |          | 
 name       | text     |           |          | 
 type       | text     |           |          | 

	id - unique 64bit integer id
	urn - text, resnet identifier for the node
	attributes - array of 'attr' id's for attributes for this entity
	name - the "Name" value from the attributes
	type - the "NodeType" value from the attributes


control - a relationship  between two or more entities
                 Table "resnet.control"
   Column   |   Type   | Collation | Nullable | Default 
------------+----------+-----------+----------+---------
 id         | bigint   |           | not null | 
 inkey      | bigint[] |           |          | 
 outkey     | bigint[] |           |          | 
 attributes | bigint[] |           |          | 

	id - unique 64bit integer id
	inkey - array id node id's for the "in" entity of the relationship
	inoutkey - arraya of node id's for the "in-out" entity of the relationship
	outkey - array of node id's for the "out" entity of the relationship 
	attributes - array of 'attr' id's for attributes for this entity


pathway - curated pathways
                 Table "resnet.pathway"
   Column   |   Type   | Collation | Nullable | Default 
------------+----------+-----------+----------+---------
 id         | bigint   |           | not null | 
 name       | text     |           |          | 
 type       | text     |           |          | 
 urn        | text     |           |          | 
 attributes | bigint[] |           |          | 
 controls   | bigint[] |           |          | 


	id - unique 64bit integer id
	name - text name of the pathway
	type - text type of the pathway
	urn - text resnet identifier
	attributes - array of attr id's with attributes of the pathway
	controls[] array of control id's for the elements of the pathway
